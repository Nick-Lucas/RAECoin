# RAECoin Contracts

## Setup

* Ensure you have `node.js` and `yarn` installed
* run `$ yarn` to install dependencies

## Usage

#### Run the test Suite

`$ yarn test`

#### Setup the app on a local ethereum network

`$ yarn start`
